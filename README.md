# ryk-Blog
> A blog site for my client's thoughts, ideas, and views

# Technologies Used
> React (w/ React-Router), Node.js/Express, MongoDB/Mongoose, Sass 

# Starting #
  From the root directory:
    1. To start the client
      > npm run build
    2. To start the server
      > npm run start
    
    
# Installing Dependencies #
  From the root directory:
    > npm install
